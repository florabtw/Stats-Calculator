Library for Pro & Free versions of the Stats Calculator app for Android.

Necessary libraries:
1. <a href="https://bitbucket.org/nkcrpn/mvplib">MVPLib</a>
2. <a href="http://developer.android.com/tools/support-library/setup.html">android-support-v7-appcompat</a>
3. <a href="http://code.google.com/p/mockito/downloads/detail?name=mockito-all-1.9.5.jar">Mockito</a>
4. <a href="https://github.com/robolectric/robolectric">Robolectric</a>
